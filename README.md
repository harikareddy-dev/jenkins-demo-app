# Jenkins Demo App

## Overview
This project demonstrates a CI/CD pipeline using Jenkins and Docker.

## Technologies Used
- Jenkins
- Docker
- Git
- GitHub
- Nginx

## Pipeline Stages
1. Checkout source code
2. Build Docker image
3. Stop old container
4. Remove old container
5. Run new container
6. Deploy application

## Result
Application deployed successfully and accessible on:
http://localhost:8085
